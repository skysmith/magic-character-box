# Enclosure Notes

Start ugly and working.

The first printable case model lives in [`cad/`](../cad/README.md). It is a screw-top rounded box with side speaker holes, an NFC target ring, Pi standoffs, a MAX98357A amp pad, and a PN532 tray under the lid.

## Prototype Box

- Cardboard, wood, or a project box is enough.
- Mark a clear "place character here" circle over the NFC reader.
- Keep USB power reachable.
- Do not expose loose battery wiring.
- Avoid metal near the reader.

## Later Box

- Speaker grille.
- Chunky play/pause button.
- Better cable management.
- Optional handle.
- Adult-access panel for the Pi and SD card.

## Paused Print Option

A paused-print enclosure that traps components inside is possible later, but the screw-top case should come first. The NFC reader, amp, speaker, and Pi will probably need adjustment after the first test fit.
